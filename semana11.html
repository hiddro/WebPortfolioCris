Practicando con Arduino


Al iniciar la clase el profesor nos pidio que descomprimieramos el archivo del software Arduino, para empezar a hacer las actividades de clase. Debido a que el profesor nos envió el archivo en formato zip tuve algunas complicaciones al momento de descomprimirlo ya que no contaba con un software para dicho caso, de la misma manera para poder colocarlo dentro de una carpeta.

Una ves descargado, ingrese al software arduino y lo primero que hicimos fue configurar ciertos puertos para el correcto uso del arduino al realizar el primer ejercicio.

A pesar que realize la configuracion y el código correctamente, no entendia porque motivo no se encendia el LED rojo que estaba conectado al arduino, después de varios intentos le consulte al profesor cual era el problema y durante el descanso me ayudo a solucionar el problema hasta que se logro el resultado final.

Regresando del refrigerio el profesor nos dejo un segundo ejercicio en el cual teniamos que conectar cables de una placa al arduino los cuales me tomo mucho tiempo poder conectarlo para poder hacer el ejercicio de la manera adecuada.
