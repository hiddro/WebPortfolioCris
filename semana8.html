ShortCuts - Atajos


  • Alt + Click(Derecho) = Mover el Objeto
  • Ctrl = Zoom in o Zoom out, para acercar y alejar el objeto
  • F = Para centrar el objeto en el espacio
  • Shift + Click(Derecho) = Movimientos encajados por cada vista
  • Click(Izquierdo) = Para modificar cosas dentro de la superficie
  • Shift + Click(Izquierdo) = Es para suavizar la imagen, se encoje los pixeles para ir boleándolo

En la barra superior se puede apreciar todas las modificaciones que se van haciendo sobre el objeto escogido.

  • Tecla X = Es para darle simetría al boleado de la imagen, es decir para que todo sea uniforme

En la parte superior izquieda podemos agrandar o achicar la brocha a nuestro gusto para trabajar correctamente

  • Shift + F = Muestra la malla para ver las divisiones que tiene el Objeto

El suavizado depende de la cantidad de polígonos que nosotros le pongamos a nuestro Objeto

Con el botón que se encuentra en la esquina se ancla para empezar a modelar y que se quede fijo el Objeto

  • Tecla W = Sirve para modificar el objeto escogido e ir acomodándolo poco a poco hasta que quede exacto

Para eliminar una máscara se presiona Ctrl y afuera del espacio para poder asi eliminarse

  • Ctrl + S = Sirve para guardar el archivo que estemos trabajando en el momento

Canvas se le llama a toda la parte donde se encuentra el modelo en 3D, es decir la parte trabajable o navegable
